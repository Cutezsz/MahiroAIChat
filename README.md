# MahiroAIChat

我创建了一个基于 Qwen-1.8B 大语言模型的 AI，人设来源于《别当欧尼酱了》中的绪山真寻（Oyama Mahiro），她拥有基础的聊天能力和特殊的语音转文字（STT）功能，并且通过在线的 NapCat 项目，可以简单地回复文本 QQ 消息。当本机项目运行时，你可以通过链接 https://ecf2951cba4c91bca6.gradio.live/ 访问或添加 QQ 号为 3453211161 的用户 [AI]MahiroAIChat 与她聊天。

本仓库不予展示模型下载，仅用于储存源代码。

Using the Qwen-1.8B LLM, I created my own AI based on the prompt of Oyama Mahiro from ACG. It has basic chatting abilities and special STT (Speech to Text); it can even reply to QQ messages simply through the online NapCat project. While the local is running, you can use it by the link https://ecf2951cba4c91bca6.gradio.live/ or chat on QQ 3453211161 named [AI]MahiroAIChat.

This repository does not show the ways of downloading the model, and it is only used to store the code.
